# vue-admin-template-layout

在 vue-admin-template-permission-control 的基础上，根据业务需求，做了以下处理：

+ 支持横向菜单，以及 setting.js 添加布局模式配置

+ 调整 tags-view 组件样式，向 vue-admin-beautiful 风格靠近

+ 引用 vuescroll 插件，优化全局滚动条效果
